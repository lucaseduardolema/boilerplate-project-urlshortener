Microserviço Encurtador de URL

  1. Insira uma URL http ou https válido
  2. Será retornado um JSON com a propiedade short_url
  3. Acesse [project_url]/api/shorturl/short_url
  4. Será redirecionado para o site
URL Shortener

Exemplo de uso:

POST [project_url]/api/shorturl - https://www.google.com

GET [project_url]/api/shorturl/392395

Será redirecionado para:

https://www.google.com
